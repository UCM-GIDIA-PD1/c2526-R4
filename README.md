# Steam Predictor

## Objetivo: Predicción de Éxito de Videojuegos 
Este proyecto consiste en el desarrollo de una plataforma de consultoría automática diseñada para ayudar a desarrolladores de videojuegos a predecir el impacto de su presencia en Steam y redes sociales antes del lanzamiento oficial.
